## CSS color Property
#### Example
#### Set the text-color for different elements:
>body {
  color: red;
}   

#### Set the text color with a HEX value:

>body {color: #92a8d1;}  

### Set the text color with an RGB value:

>body {color: rgb(201, 76, 76);}
